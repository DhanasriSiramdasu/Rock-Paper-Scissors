# ✊✋✌️ Rock Paper Scissors Game

A fun and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**. Play against the computer, which randomly selects its move, while the game automatically determines the winner, updates the score, and displays the result.

---

## 🚀 Features

- 🎮 Play against the computer
- 🤖 Random computer move generation
- 🏆 Automatic winner determination
- 📊 Live score tracking for Player and Computer
- 🎨 Dynamic result message with color indication
- 📱 Clean and responsive user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 How to Play

1. Click on **Rock**, **Paper**, or **Scissors**.
2. The computer randomly selects its choice.
3. The winner is determined based on the game rules:
   - ✊ Rock beats Scissors
   - ✋ Paper beats Rock
   - ✌️ Scissors beats Paper
4. The result is displayed instantly.
5. The score updates automatically after every round.

---

## ⚙️ Game Logic

- Generates a random choice for the computer.
- Compares the player's choice with the computer's choice.
- Displays:
  - 🟢 Green for a win
  - 🔴 Red for a loss
  - 🔵 Blue for a draw
- Updates the player and computer scores after each round.
